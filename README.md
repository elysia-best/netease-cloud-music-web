# 网易云音乐 - Web 版本

这是一个基于 Electron 的网易云音乐 Web 客户端项目。该项目旨在提供一个桌面应用程序来访问网易云音乐服务。

## 功能特点

- 使用 Electron 构建的桌面应用程序
- 集成了网易云音乐的 Web 界面
- 支持系统托盘操作
- 跨平台支持（Windows, macOS, Linux）

## 技术栈

- Electron
- JavaScript
- HTML/CSS

## 安装指南

1. 确保你的系统上安装了 [Node.js](https://nodejs.org/) 和 npm。
2. 克隆仓库到本地：
   ```bash
   git clone https://gitee.com/qinyinan/netease-cloud-music-web.git
   ```
3. 进入项目目录并安装依赖：
   ```bash
   cd netease-cloud-music-web
   npm install
   ```
4. 启动应用：
   ```bash
   npm start
   ```

## 使用说明

启动应用后，你将看到一个简单的界面来访问网易云音乐。你也可以通过系统托盘来控制应用。
